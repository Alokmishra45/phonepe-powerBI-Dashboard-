# phonepe-powerBI-Dashboard-
Power BI dashboard analyzing phonePe payment transactions 


## Project Overview

An interactive Power BI dashboard analyzing PhonePe digital payment transactions across India.
The goal was to identify transaction trends, user behavior, and service-level performance.

---

## Problem Statement

PhonePe processes millions of transactions daily.
The goal of this project was to analyze payment data to answer:
- Which service generates the highest transaction value?
- When do users transact the most — weekdays or weekends?
- Which age group is the most active?
- What is the overall payment success rate?

## Dashboard Features

- KPI Cards — Total Transactions, Total Value, Unique Users, Success Rate
- Transactions Over Time — Monthly trend line (Jan–Dec)
- Age Segment Contribution — Donut chart by user age groups
- Service Transaction Value Analysis — Loans vs Insurance comparison
- Top 5 Users — By total transaction value
- Weekday vs Weekend Usage — 71.54% weekday dominance
- Payment Status Filter — Failed / Pending / Successful

---

## Key Insights

- Weekdays account for 71.54% of all transactions
- Loans is the highest value service (2.5 Billion)
- Age group 37.4% is the largest user segment
- Transaction volume peaks mid-year (Jun–Aug)
